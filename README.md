一、原理
基于面向对象编程的思想，采用了面向对象式的绘画原理，整体上分为Figure->Axes->Chart(hist,bar,barh,scatter,pie,plot...)
为了便于理解，Figure看作为窗口/画板，Axes看作为窗口中的一个区域/画纸（画纸的骨架），Chart看作为画在画纸上的各种图表。从而可以综合理解：可以先有一张画板，在画板上铺上几张画纸，从而在各张画纸上分别绘画各种图表。
